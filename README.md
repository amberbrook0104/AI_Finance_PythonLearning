# AI Finance Python Learning

## 目标
系统学习 Python、AI 与金融，并记录学习过程和项目经验。

## 仓库结构
- `/01_Python_Basics` - Python基础知识练习
- `/02_Data_Analysis` - 数据分析工具与练习
- `/03_AI_Finance` - AI在金融中的实战示例
- `/04_Projects` - 项目实战
- `/05_Notes` - 笔记与阅读日志
- `/06_Resources` - PDF教材与数据集

## 使用方法
1. 每学完一个知识点或章节，更新对应 Notebook 文件。
2. 将阅读笔记保存到 `/05_Notes`。
3. 每次修改提交 commit，并写明修改内容。
4. 周期性 Push 到 GitHub 云端。
